# Installation

## FinX Python SDK

FinX Python SDK is an installable python package registered with PyPi.org and installable as follows:

#### Linux Install

##### latest release

```bash
pip install finx-io
```

#### Mac Install

```mac command line
python3 -m pip install finx-io```

##### beta release

```bash
pip install -i https://test.pypi.org/simple/ finx-io
```
